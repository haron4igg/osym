Лабараторная работа №2

Определение конических сечений, удовлетворяющих заданным условиям
непрерывности и касания.

Этот метод основан на применении классических методов теории конических сечений к вопросу проектирования поперечных сечений фюзеляжа самолета.

Если два конических сечения задаются уравнениями:

S1(x, y) = 0 и S2(x, y) = 0 (2.1)

и эти сечения пересекаются в 4 точках, то уравнение

(1 - k)S1 + kS2 = 0 (2.2)

определяет при любых k конические сечения, проходящие через точки пересечения сечений (2.1). При варьировании k образуется семейство конических сечений. Для определения величины k задается еще 1 точка (не являющаяся т. пересечения). Если это точка (x0, y0), то из (2.2) следует:

k = S1(x0, y0) / (S1(x0, y0) - S2(x0, y0)) (2.3)

Пусть 2 прямые задаются уравнениями:

Тогда уравнение

l1 l2 = 0

определяет вырожденное коническое сечение, получаемое в результате рассечения конуса плоскостью, проходящей через его вершину параллельно оси. Такие пары прямых можно использовать для определения невырожденных конических сечений при помощи уравнения вида (2.2):

( 1 - k) l1l2 + kl3l4 = 0 (2.4)

Если существуют 4 т. пересечения двух пар прямых линий (l1,l2);(l3,l4), то, задавая произвольную пятую точку, можно определить значение k по формуле вида (2.3). Найдем коническое сечение, имеющее 2 заданные касательные в двух точках и проходящее через третью заданную точку. Если прямые l3=0 и l4=0 совпадают, то уравнение

(1-k) l1l2 + kl = 0 (2.5)

определяет коническое сечение, задаваемое четырьмя точками: двумя точками касания A, D ( l1-касательная в точке A, l2-касательная в т. D), точкой пересечения касательных E, а 4-ая точка F, называемая  опорной точкой 0, определяет параметр k.

Если F делит пополам отрезок, соединяющий середины отрезков DE и AE, то коническое сечение является параболой, называемой  пропорциональной кривой. Если F находится между этой параболой и AD, то получаем эллипс, если F за пределами параболы- гиперболу. В предложенном Лаймингом (Liming, 1944) методе проектирования поперечного сечения фюзеляжа самолета конфигурация каждого сечения определяется 5 точками. Поскольку каждое сечение симметрично относительно вертикальной оси, можно проектировать только половину сечения, причем касательные в точках E и A должны быть горизонтальными. Поскольку т. C является точкой максимальной ширины сечения, касательная в C должна иметь вертикальное направление. Проектируемое сечение состоит, таким образом, из двух конических сечений с общей касательной в т.C. Фюзеляж в целом описывается пятью кривыми- следами точек A,B,C,D,E.

Задание для самостоятельной работы.

1. Изменяя параметр k от 0 до 1, построить гладкий переход от кругового сечения к квадратному. В качестве S1 взять четверть окружности, а S2- 2 перпендикулярных радиуса этой окружности.

2. Выбрать 2 пары пересекающихся прямых: (l1,l2),(l3,l4). Варьируя пятую точку построить семейство кривых, проходящих через 4 точки пересечения. По дискриминанту определить граничную параболу, зоны эллипсов и гипербол.

3. Рассмотреть случай, когда l3 совпадает с l4. Определить положение опорной точки, построить пропорциональную кривую, определить зоны эллипсов и парабол.

4. Методом Лайминга спроектировать поперечное сечение фюзеляжа, варьируя положение точки максимальной ширины и высоту.
